# Introduction
Thank you for considering contributing to the `queeems` molecular sequence
exploratory R (and prospective Bioconductor) package. Your input will
definitely be treated with great importance.

# Proposing an update
Should you wish to propose an update to the package, kindly raise an issue
on the dedicated [Github page](https://github.com/thsadiq/queeems/issues).
Rest assured that your proposal will be seriously considered and all valid
concerns will be addressed promptly.

# Reporting issues or problems with the package
Bugs and/or issues may be reported by creating an issue on the package's
[Github page](https://github.com/thsadiq/queeems/issues).

# Seeking support
Users that need assistance using the package or any of the functions
therein should please consult the function description file embedded
with the package. Detailed examples are also available in the package's
[vignettes](vignettes) folder. Further assistance may be sought by sending
an e-mail directly to the corresponding author,
[Hassan Sadiq](hassan.t.sadiq@gmail.com).
